5.24. Grep-2.16

The Grep package contains programs for searching through files.

预计构建时间: 0.2 SBU
盘空间需求: 21 MB

5.24.1. 安装 Grep

准备编译 Grep： 

./configure --prefix=/tools

编译软件包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中为临时工具运行测试套件。如果无论如何都要运行 Grep 的测试套件，执行以下命令： 

make check

安装软件包： 

make install

这个包的详细信息位于 第 6.31.2 节 “Contents of Grep.”
