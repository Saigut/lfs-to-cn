5.28. Patch-2.7.1

The Patch package contains a program for modifying or creating files by applying a “patch” file typically created by the diff program.

预计构建时间: 0.1 SBU
盘空间需求: 3.4 MB

5.28.1. 安装 Patch

准备编译 Patch： 

./configure --prefix=/tools

编译软件包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中为临时工具运行测试套件。如果无论如何都要运行 Patch 的测试套件，执行以下命令： 

make check

安装软件包： 

make install

这个包的详细信息位于 第 6.55.2 节 “Contents of Patch.”
