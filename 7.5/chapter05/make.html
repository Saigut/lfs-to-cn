5.27. Make-4.0

The Make package contains a program for compiling packages.

预计构建时间: 0.1 SBU
磁盘空间需求: 11.2 MB

5.27.1. 安装 Make

准备编译 Make： 

./configure --prefix=/tools --without-guile

配置选项含义：

--without-guile

此项确保 Make-4.0 不链接 Guile 库，这个库现在可能在宿主系统上，但在下一章的 chroot 环境中将不可用。

编译软件包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中为临时工具运行测试套件。如果无论如何都要运行 Make 的测试套件，执行以下命令： 

make check

安装软件包： 

make install

这个包的详细信息位于 第 6.54.2 节 “Contents of Make.”
