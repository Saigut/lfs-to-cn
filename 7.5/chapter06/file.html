6.12. File-5.17

File 软件包包含一个用于检测给定文件类型的工具。

预计构建时间: 0.1 SBU
磁盘空间需求: 12.5 MB

6.12.1. 安装 File

准备编译 File： 

./configure --prefix=/usr

编译软件包： 

make

要测试结果，执行： 

make check

安装软件包： 

make install

6.12.2. File 的内容

安装的程序: file
安装的库: libmagic.so

简述

file

试图对每个给定的文件进行分类；通过进行多种测试 ——文件系统测试、魔数测试和语言测试——来实现

libmagic.so

包含用于识别魔数的途径，被 file 程序使用
