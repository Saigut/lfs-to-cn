6.18. Sed-4.2.2

Sed 软件包包含一个流编辑器。

预计构建时间: 0.2 SBU
磁盘空间需求: 6.7 MB

6.18.1. 安装 Sed

准备编译 Sed： 

./configure --prefix=/usr --bindir=/bin --htmldir=/usr/share/doc/sed-4.2.2

新的配置选项的含义：

--htmldir

此项设置将要安装 HTML 文档的目录。

编译软件包： 

make

生成 HTML 文档： 

make html

要测试结果，执行： 

make check

安装软件包： 

make install

安装 HTML 文档： 

make -C doc install-html

6.18.2. Sed 的内容

安装的程序: sed
安装的目录: /usr/share/doc/sed-4.2.2

简述

sed

通过一次简单的操作筛选和改变文本文件
