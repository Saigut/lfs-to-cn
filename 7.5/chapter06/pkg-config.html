6.20. Pkg-config-0.28

pkg-config 包包含一个工具，在 configure 和 make 文件执行时传递 include 路径和/或库路径给构建工具。

预计构建时间: 0.4 SBU
磁盘空间需求: 31 MB

6.20.1. 安装 Pkg-config

准备编译 Pkg-config： 

./configure --prefix=/usr         \
            --with-internal-glib  \
            --disable-host-tool   \
            --docdir=/usr/share/doc/pkg-config-0.28

新 configure 选项的含义：

--with-internal-glib

此项允许 pkg-config 使用它的内部版本的 Glib，因为 LFS 中外部版本是不可用的。

--disable-host-tool

此项禁止一个不期望的到 pkg-config 程序的硬链接的创建。

编译软件包： 

make

要测试结果，执行： 

make check

安装软件包： 

make install

6.20.2. Pkg-config 的内容

安装的程序: pkg-config
安装的目录: /usr/share/doc/pkg-config-0.28

简述

pkg-config

为指定的库或包返回元信息。
