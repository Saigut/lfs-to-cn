6.30. Bison-3.0.2

Bison 程序包包含分析器生成器。

预计构建时间: 0.3 SBU
磁盘空间需求: 31 MB

6.30.1. 安装 Bison

准备编译 Bison： 

./configure --prefix=/usr

编译软件包： 

make

要测试结果 (大约 0.5 SBU)，执行： 

make check

安装软件包： 

make install

6.30.2. Bison 的内容

安装的程序: bison 和 yacc
安装的库: liby.a
安装的目录: /usr/share/bison

简述

bison

以一系列规则创建用于分析文本结构的程序；是 Yacc (Yet Another Compiler Compiler) 的替代

yacc

bison 的封装，针对仍然调用 yacc 而不是 bison 的程序；它调用 bison 并传递 -y 选项

liby.a

Yacc 库包含与 Yacc 兼容的 yyerror 和 main 函数的实现；此库通常不会很常用，但 POSIX 要求其存在
