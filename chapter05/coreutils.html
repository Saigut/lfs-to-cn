5.18. Coreutils-8.22

The Coreutils package contains utilities for showing and setting the basic system characteristics.

预计构建时间: 0.8 SBU
磁盘空间需求: 133 MB

5.18.1. 安装 Coreutils

准备编译 Coreutils： 

./configure --prefix=/tools --enable-install-program=hostname

配置选项含义：

--enable-install-program=hostname

此项启用构建和安装 hostname 二进制文件––它默认禁用，但 Perl 的测试套件需要。

编译软件包： 

make

编译现在完成了。前面讨论过，没有必要在这一章中为临时工具运行测试套件。如果无论如何都要运行 Coreutils 的测试套件，执行以下命令： 

make RUN_EXPENSIVE_TESTS=yes check

RUN_EXPENSIVE_TESTS=yes 参数告诉测试套件运行几个额外的测试，它们在某些平台上相对昂贵 (CPU 运算能力和内存使用)，但是在 Linux 上通常不是问题。

安装软件包： 

make install

这个包的详细信息位于 第 6.26.2 节 “Contents of Coreutils.”
